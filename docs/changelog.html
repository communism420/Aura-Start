Release notes

Aura Start 1.2.0

Migration and Data Ownership Release. This release focuses on safer data movement, clearer recovery, and a more keyboard-friendly workflow.

Highlights

  • Improved A Fine Start migration flow.
  • Added first-run onboarding, a better empty state, and opt-in demo data.
  • Added a visible Export / Backup hub.
  • Added Restore Points Manager.
  • Added Privacy Promise.
  • Improved search and keyboard navigation.
  • Added keyboard shortcuts help.
  • Added Command Palette.
  • Added Duplicate Finder with user-confirmed deletion.
  • Improved data-loss protection for destructive actions.

Documentation

The user documentation now includes migration and comparison guides, privacy information, screenshots, and clearer explanations of export and recovery.
