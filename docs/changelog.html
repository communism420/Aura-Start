Release notes

Aura Start 2.0.0

Migration, personalization, Firefox, and data ownership release. This release focuses on nested organization, safer recovery, optional customization, cross-browser packaging, and keyboard-friendly workflows.

Highlights

  • Added nested groups up to two levels, including parent selection and drag-and-drop reparenting.
  • Added Fuse.js-powered fuzzy search across title, URL, description, and tags, with result counts and quick filters.
  • Reworked restore points into a searchable Restore Timeline with day grouping, action filters, export, and confirmed restore.
  • Added optional Save open tabs workflow with runtime tabs permission, preview, duplicate filtering, and restore point protection.
  • Added built-in and custom local background images with blur, dim, and positioning controls.
  • Added optional widgets for header clock, local Markdown notes, and a Pomodoro timer.
  • Added Firefox-oriented build support with WebExtension API wrappers and Device OAuth fallback for Google Drive sync.
  • Added stronger A Fine Start migration flow with safer import preview and compatibility notes.
  • Added first-run onboarding, empty state actions, demo data, and Google Drive restore during setup.
  • Added Export / Backup hub, Command Palette, Duplicate Finder, and Privacy Promise.
  • Fixed Google Drive sync behavior across test and Chrome Web Store builds, including OAuth flow, reconnect states, token persistence, and automatic backup handling (I hope).
  • Kept Google Drive sync optional and limited to the hidden appDataFolder storage used by Aura Start.
  • Improved keyboard shortcuts, Cyrillic-layout shortcut handling, search focus behavior, and Command Palette access.
  • Improved destructive-action safety with restore points, confirmations, and clearer recovery messaging.
  • Updated translations across all supported languages and fixed untranslated UI labels.
  • Redesigned the documentation website and refreshed privacy, store listing, and release materials.
  • Updated screenshots to match the new Aura Start design.

Documentation

The user documentation now includes migration and comparison guides, privacy information, screenshots, and clearer explanations of export and recovery.
