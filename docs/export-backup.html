User guide

Export / Backup

Aura Start is export-first. Exports are created locally in your browser and are not uploaded to an Aura Start server.

Available formats

  • Full Backup JSON: best for restoring Aura Start data later.
  • Browser Bookmarks HTML: useful for importing into browser bookmark managers.
  • Markdown: readable grouped lists for notes or documentation.
  • CSV: spreadsheet-friendly archive format.
  • A Fine Start-compatible export code: useful if you want a basic grouped-link format again later.

When to use Full Backup JSON

Use Full Backup JSON before major imports, resets, duplicate cleanup, or any change where you want an independent file backup. It is the format designed for restoring Aura Start's own data.

A Fine Start-compatible export

The compatibility format is intentionally basic. It can preserve grouped link names and URLs, but Aura Start-specific fields such as descriptions and tags may not be included if the compatibility format does not support them.

Empty or partial data

Export works even when your start page is empty. If data is incomplete or corrupted, use the recovery screen or restore points before replacing your current local data.

Exports are local files. Keep backups somewhere you control.
