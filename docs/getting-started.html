User guide

Getting Started

Aura Start replaces your browser new tab page with clean nested groups of links. It works locally by default and does not require an account.

Install from source

Until the Chrome Web Store listing is available, Aura Start can be installed manually from the project repository.

  1. Open the Aura Start repository on GitHub.
  2. Follow the build instructions in the README.
  3. For Chrome or Chromium testing, open chrome://extensions, enable Developer mode, and load the generated Chrome build folder.
  4. For Firefox testing, use about:debugging and load the generated Firefox build manifest.

Choose how to start

On first run, Aura Start can start fresh, restore an existing hidden Google Drive sync file, import from A Fine Start, or import a backup file. If you choose Google Drive restore and no sync file exists, Aura Start shows a clear error and leaves local data unchanged.

Demo groups are available from the empty state after explicit user action. Aura Start does not add demo data automatically.

Create groups and links

  • Create groups for areas like Work, Reading, Tools, or Research.
  • Create nested groups when one level of structure helps, such as Research -> Deep dives.
  • Add links with titles, URLs, optional descriptions, and optional tags.
  • Use edit mode when you want to reorder groups or links with drag and drop, including moving a group into or out of a parent group.
  • Use compact mode or configurable columns when you want a denser layout.

Find links quickly

Search can match link title, URL, description, and tags. It uses fuzzy matching for small typos, keeps parent groups visible when a nested child matches, shows result counts, and can be narrowed with quick filters.

Command Palette is available from the Aura Start UI. Ctrl+K or Cmd+K opens it when Chromium assigns that extension shortcut to Aura Start; if the shortcut does not work, check chrome://extensions/shortcuts. Letter shortcuts use physical key codes, so they continue to work on Latin and Cyrillic keyboard layouts when focus is inside the Aura Start page.

Save open tabs

If the Save open tabs feature is enabled in Settings, Aura Start can preview tabs from the current window and save them into a new group. It asks for the optional tabs permission at runtime, filters duplicates and unsupported browser pages, and creates a restore point before saving.

Personalize the page

Settings include built-in background images, custom local background upload, blur, dim, and positioning controls. Optional widgets can show a header clock, local Markdown notes, and a lightweight Pomodoro timer.

Keep backups

Use Export / Backup regularly if your start page contains important links. Full Backup JSON is the best format for restoring Aura Start data later.
