User guide

Google Drive Sync

Google Drive sync is optional and off by default. Aura Start remains usable as a local-only extension without connecting a Google account.

What sync stores

When you connect Google Drive, Aura Start stores one hidden sync file named aura-start-sync.json in Google Drive appDataFolder. This file can include your Aura Start groups, links, settings, restore points, and sync metadata needed to keep the backup current.

What Aura Start does not access

  • No full Google Drive access.
  • No reading or scanning normal visible Drive files.
  • No visible Drive backup folder.
  • No Google account use for analytics, tracking, advertising, or profiling.

Connection behavior

If a hidden Aura Start sync file already exists, Aura Start can restore it after creating a local restore point. If no sync file exists, Aura Start can create one from your current local data.

Disconnect and delete backup

Aura Start provides a confirmed action to delete the hidden Drive sync file and disconnect the Google account. This does not delete local Aura Start data and does not touch normal Drive files.

Use local backups too

Google Drive sync is convenient, but Full Backup JSON is still the clearest portable backup format. Keep local file backups for important data.
