Product

Everything important stays visible and recoverable.

Aura Start is built for people who want a calm new tab, clear data ownership, and practical recovery tools instead of account lock-in.

01

Clean link groups

Organize links into readable groups with descriptions, tags, search modifiers, themes, compact mode, and configurable columns.

02

Import from A Fine Start

Import A Fine Start export codes for migration. Aura Start is independent and not affiliated with A Fine Start.

03

Export / Backup

Export Full Backup JSON, Browser Bookmarks HTML, Markdown, CSV, or an A Fine Start-compatible export code.

04

Restore Points

Review and restore local safety snapshots created before destructive changes where available.

05

Command Palette

Open common actions from the visible palette button or Ctrl+K/Cmd+K when the browser assigns that shortcut.

06

Duplicate Finder

Scan for exact and possible duplicate links. Deletion remains user-selected and confirmed.

07

Optional Drive backup

Google Drive sync is optional, restores or creates Aura Start's hidden sync file, and syncs local changes automatically after connection.

08

Privacy Promise

No ads, no backend, no telemetry, no browser history permission, and no browser bookmarks permission.

Data ownership

Your links belong to you.

Aura Start stores your groups, links, settings, and restore points locally by default. Exports are created locally in your browser, so you can keep your own backups and move your data when you need to.

Migration

A careful path from A Fine Start.

Aura Start supports importing A Fine Start export codes and exporting an A Fine Start-compatible code for the basic grouped-link format. It does not claim affiliation with A Fine Start.

Privacy-first

No account, no analytics, no tracking.

Aura Start has no backend and does not request browser history, bookmarks, cookies, tabs, webRequest, scripting, or full Google Drive access.

Optional backup

Google Drive sync only when you choose it.

When enabled, Drive sync uses Chromium extension identity APIs and the drive.appdata scope to store Aura Start's hidden aura-start-sync.json file. Aura Start does not request full Drive access or visible Drive files.
