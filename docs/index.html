Open-source Chromium new tab extension

A private, local-first start page for your links — free, open-source, and easy to export.

Aura Start keeps clean groups of links on your Chromium new tab page, with local storage by default, no required account, no analytics, and export-first data ownership.

Why Aura Start?

Local-first by default

Your groups, links, settings, and restore points live in local extension storage unless you choose optional Google Drive sync.

Export anytime

Export Full Backup JSON, Browser Bookmarks HTML, Markdown, CSV, or an A Fine Start-compatible export code.

No required account

Aura Start has no backend, no ads, no analytics, and no tracking. Google Drive sync is optional and off by default.

Built for clean link groups

  • Custom groups and links with descriptions and tags.
  • Search across title, URL, description, tags, and supported query modifiers.
  • Light, dark, and system themes with compact mode and configurable columns.
  • Drag-and-drop reordering in explicit edit mode.
  • Command Palette and keyboard shortcuts for faster navigation.

Designed around recovery

  • First-run onboarding and useful empty states.
  • Optional demo groups added only after user action.
  • Export / Backup hub for local file backups.
  • Restore Points Manager for local safety history.
  • Duplicate Finder scans read-only and deletes only after selection and confirmation.

Import from A Fine Start

Aura Start supports importing A Fine Start export codes and exporting an A Fine Start-compatible code if you want a basic grouped-link format again later.

Aura Start is independent and not affiliated with A Fine Start. A Fine Start is mentioned only for inspiration, migration, and compatibility.

Optional Google Drive backup

When enabled, Google Drive sync uses Chrome identity and only the drive.appdata scope to store one hidden aura-start-sync.json file in Google Drive appDataFolder.

Aura Start does not request full Drive access and does not read visible Drive files.

Privacy Promise

No analytics or tracking

No telemetry, behavioral profiling, ads, affiliate replacement, or hidden data collection.

Least-privilege permissions

No bookmarks, history, cookies, tabs, webRequest, scripting, or full Google Drive permission.

Open-source under MIT

The source, docs, and build scripts are intended to be public and auditable.
