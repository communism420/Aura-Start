Aura Start Privacy Policy

Effective date: May 10, 2026

Aura Start is a local-first Chromium extension. It replaces the new tab page with user-created groups of links and keeps those groups under the user's control.

Aura Start does not operate a backend service, does not require an account, does not use analytics, and does not sell, rent, share, upload, or sync user data.

Scope

This Privacy Policy applies to the Aura Start Chromium extension and the local data it handles while providing its new tab, settings, import, export, and restore point features.

Data Aura Start Handles

Aura Start handles only the data needed for its single purpose:

  • Link groups created by the user.
  • Link titles, URLs, optional descriptions, optional tags, and ordering.
  • Extension settings such as theme, language, columns, compact mode, search visibility, and link-opening behavior.
  • Local restore points created before imports, resets, and destructive actions.
  • Backup files or A Fine Start export codes selected or pasted by the user for import.

Aura Start does not request browser history, browser bookmarks, tabs, cookies, web requests, identity, scripting, or host permissions.

Local Storage

Aura Start stores its data in chrome.storage.local inside the user's browser profile. In development mode only, when chrome.storage.local is unavailable, Aura Start can use browser localStorage as a local fallback.

Aura Start does not operate a server-side database. User-created groups, links, imports, exports, settings, and restore points remain local to the browser profile unless the user chooses to export them to a file.

Network, Accounts, Analytics, and Tracking

Aura Start does not require an account and does not include:

  • Analytics or telemetry.
  • Trackers, ads, affiliate link replacement, or behavioral profiling.
  • Cloud sync or remote backup services.
  • Remote APIs for its core functionality.
  • Remotely hosted extension code.

Export files are created locally in the browser using Blob downloads. Import files are read locally by the browser and are validated before they change local extension storage.

User Control

Users can create, edit, delete, export, import, reset, and restore their own Aura Start data inside the extension. Full Backup JSON export is provided so users can keep independent backups of their data in normal files.

If a user resets Aura Start data inside the extension, the extension creates a local restore point before the destructive action when restore protection is enabled. Users can also delete restore points.

Data Sharing and Sale

Aura Start does not sell user data. Aura Start does not rent, trade, share, disclose, or transfer user-created groups, links, settings, restore points, imports, or exports to third parties.

Chrome Web Store Limited Use Statement

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements. Aura Start uses Chrome extension storage only to provide and improve its single user-facing purpose: a local, private, exportable new tab start page.

Security

Aura Start bundles its extension code with the extension package and does not execute remotely hosted code. Local data remains in the user's browser profile. Users should protect their operating system account and browser profile, and should store exported backup files securely.

Open Source

Aura Start is fully open-source under the MIT License. The source code, build scripts, validation scripts, and documentation can be inspected, built, forked, and modified under that license. There is no proprietary server component or closed sync service required for the extension to work.

Changes to This Policy

This policy may be updated when Aura Start changes. Any updated policy should remain consistent with the extension's actual behavior and with the privacy disclosures provided in the Chrome Web Store Developer Dashboard.

Contact

For privacy or security questions, use the support contact listed on the Chrome Web Store listing or the project repository where Aura Start is published.
