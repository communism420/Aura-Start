Restore Points

Restore points are local safety snapshots. Aura Start uses them to make destructive changes easier to recover from.

When restore points are useful

  • Before replacing data during import.
  • Before restoring from Google Drive during connection.
  • Before reset or destructive cleanup actions where supported.
  • Before duplicate deletion when you confirm selected items.

Restore Points Manager

The Restore Points Manager shows local snapshots with their reason, date, and data counts when available. You can restore a snapshot or export it as a JSON backup.

Before restoring

Restoring changes your current local Aura Start data. Aura Start creates a new restore point for the current state before restore where possible, so you have a way back if the restored snapshot is not the one you wanted.

Good habits

Restore points are helpful, but they are still local browser data. Keep a Full Backup JSON file somewhere safe before very large changes.

For file backups, use Export / Backup.
