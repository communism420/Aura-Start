Store Asset Checklist

Maintainer-only document. This page is for project release preparation and is not needed for normal Aura Start users.

This folder tracks the Chrome Web Store and promotional screenshot set. The final PNG files are captured from the real Aura Start 1.2.0 UI and stored in Photo/, with GitHub Pages copies in docs/assets/screenshots/.

The public GitHub Pages site uses the *-20260524.png screenshot copies to avoid stale cached images after design refreshes.

Capture rules:

  • Use current Aura Start 1.2.0 UI.
  • Do not use drawn mockups that differ from the real extension design.
  • Do not show personal data, OAuth tokens, email addresses, or browser profile details.
  • Do not use A Fine Start screenshots or logos.
  • Do not claim Chrome Web Store availability before the listing is live.
