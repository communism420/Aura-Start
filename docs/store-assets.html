Store Asset Checklist

Maintainer-only document. This page is for project release preparation and is not needed for normal Aura Start users.

This folder tracks the Chrome Web Store, Firefox Add-ons, and public site screenshot set. The final store PNG files are captured from the real Aura Start 2.0.0 UI and stored in Chrome Submit/Photo/ and Firefox Submit/Photo/, with public site copies in docs/assets/screenshots/.

The public site uses the *-20260630.png screenshot copies to avoid stale cached images after the nested groups, fuzzy search, backgrounds, widgets, Save tabs, and Restore Timeline refresh.

Capture rules

  • Use current Aura Start 2.0.0 UI.
  • Do not use drawn mockups that differ from the real extension design.
  • Do not show personal data, OAuth tokens, email addresses, or browser profile details.
  • Do not use A Fine Start screenshots or logos.
  • Do not claim Chrome Web Store or Firefox Add-ons availability before the listing is live.
